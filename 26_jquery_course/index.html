PICT B.E. Course Structure

First Year (FE)

  • Engineering Mathematics-I: Foundation of calculus and algebra.
  • Engineering Physics: Basics of mechanics, optics, and waves.
  • Engineering Chemistry: Chemical reactions and applications in engineering.
  • Programming in C: Introduction to coding with C language.
  • Basic Electrical Engineering: Fundamentals of circuits and machines.

Second Year (SE)

  • Data Structures: Arrays, stacks, queues, trees, graphs.
  • Computer Organization: CPU architecture, memory, instruction sets.
  • OOP in Java: Object-oriented principles using Java.
  • Digital Electronics: Logic gates, combinational and sequential circuits.
  • Discrete Mathematics: Logic, sets, relations, and functions.

Third Year (TE)

  • Operating Systems: Process management, memory, file systems.
  • Database Management Systems: SQL, normalization, indexing.
  • Computer Networks: OSI model, protocols, routing.
  • Software Engineering: SDLC, agile methods, UML.
  • Web Technology: HTML, CSS, JavaScript, and backend basics.

Final Year (BE)

  • Machine Learning: Algorithms, regression, classification.
  • Information and Cyber Security: Threats, encryption, security models.
  • Cloud Computing: Virtualization, IaaS/PaaS/SaaS, deployment models.
  • Big Data Analytics: Hadoop, Spark, data mining techniques.
  • Project Work: Final year major software/hardware project.
